# PRJ TRAP VAULT

This repository contains the PRJ TRAP VAULT storefront application.

## Getting Started

Run the development server locally:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Project Notes

- The site is built to be responsive across mobile, tablet, and desktop.
- Authentication and checkout functionality are currently disabled until configuration is completed.
- The site uses your custom brand assets and logo for the favicon.

## Deployment

This app can be deployed to any static-compatible hosting platform.
